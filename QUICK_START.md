# 🚀 QUICK START GUIDE - CV SEMASA HIDUP Website

## ⚡ Mulai dalam 3 Langkah Mudah

### **Langkah 1: Download Files**
✅ Download folder dengan:
- `index.html` - File website utama
- `logo.png` - Logo perusahaan
- `README.md` - Dokumentasi lengkap

### **Langkah 2: Buka di Browser**
Pilih salah satu:

**Cara 1 - Langsung**
```
1. Double-click file `index.html`
2. Browser otomatis membuka website
3. Done! ✅
```

**Cara 2 - Drag & Drop**
```
1. Buka browser favorit (Chrome, Firefox, Edge, dll)
2. Drag file `index.html` ke browser window
3. Website langsung tampil ✅
```

**Cara 3 - Right Click**
```
1. Right-click `index.html`
2. Pilih "Open with" → Browser pilihan Anda
3. Website terbuka ✅
```

### **Langkah 3: Nikmati Website!**
- Jelajahi 4 halaman utama
- Klik navigasi untuk berpindah halaman
- Isi form kontak jika ingin berinteraksi
- Semua halaman integrated dalam 1 file HTML

## 🔧 Customization Mudah

### **Mengubah Teks**
Cari di `index.html` dan edit langsung:
```html
<h1>Solusi Teknologi Terpadu untuk <span class="highlight">Bisnis Modern</span></h1>
<!-- Ubah teks di atas sesuai keinginan -->
```

### **Mengubah Nomor Telepon/Email**
Cari text seperti:
```html
<p>089532559611</p>
<p>cvsemasahidup@gmail.com</p>
<!-- Ganti dengan nomor/email Anda -->
```

### **Mengubah Warna Brand**
Cari di bagian CSS (awal file):
```css
:root {
    --primary: #1a4d7a;     /* Warna biru utama */
    --secondary: #e8931f;   /* Warna oranye */
    --accent: #00b4d8;      /* Warna cyan */
}
/* Ganti nilai hex color sesuai brand Anda */
```

### **Mengganti Logo**
1. Siapkan logo baru (PNG atau JPG)
2. Rename menjadi `logo.png`
3. Tempat di folder yang sama dengan `index.html`
4. Selesai! Logo otomatis tampil di navbar & header

## 🌐 Deploy Website (Upload Online)

### **Option 1: GitHub Pages (GRATIS, Recommended)**
1. Buat akun di github.com
2. Buat repository baru
3. Upload files (index.html, logo.png, README.md)
4. Settings → Pages → Select main branch
5. Website live di: `yourname.github.io/repo-name` ✅

### **Option 2: Netlify (GRATIS)**
1. Buka netlify.com
2. Drag & drop folder ke site
3. Website instant live dengan domain free ✅
4. Custom domain bisa ditambah nanti

### **Option 3: Hosting Berbayar**
- Gunakan Hostinger, Bluehost, atau hosting lainnya
- Upload via FTP atau file manager
- Configure domain
- Website siap diakses publik ✅

## 🎓 Troubleshooting Cepat

| Problem | Solusi |
|---------|--------|
| **Logo tidak muncul** | Pastikan file `logo.png` ada di folder yang sama |
| **Halaman tidak berpindah** | Refresh browser (F5 atau Ctrl+R) |
| **Text terlihat blur** | Zoom out browser (Ctrl + Minus) atau zoom in (Ctrl + Plus) |
| **Form tidak berfungsi** | Form demo hanya show alert, tidak kirim email |
| **Warna berbeda dari design** | Clear cache (Ctrl+Shift+Delete) atau gunakan incognito mode |
